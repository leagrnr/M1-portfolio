## Description

Application web interactive affichant sur une carte Mapbox des indicateurs clés par pays — PIB, population, espérance de vie, taux d'alphabétisation, etc. — en temps réel via l'API World Bank.

L'utilisateur peut explorer le monde visuellement, cliquer sur un pays et accéder instantanément à ses données économiques et démographiques actualisées.

Stack : JavaScript · Mapbox GL JS · World Bank API REST
